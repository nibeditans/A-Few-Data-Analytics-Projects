Which Approach is better?

  • Option 1: df.drop_duplicates() (Full Row Deduplication)
  • Option 2: drop_duplicates(subset=[...key columns...])

Let's do both:

In[17]:
# Version 1: Full data with only exact duplicates removed
df_deduped = df.drop_duplicates()

# Version 2: Highly unique EV configs only
df_unique_profiles = df.drop_duplicates(subset=[
    'model_year', 'make', 'model', 'ev_type',
    'electric_range', 'cafv_eligibility', 'county', 'city'
])

We'll use:

  • df_deduped for insights, trends, volume-based visuals.
  • df_unique_profiles for patterns, relationships, distributions.
